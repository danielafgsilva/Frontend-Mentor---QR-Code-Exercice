# QR code component · Frontend Mentor

Solução para o desafio [QR code component](https://www.frontendmentor.io/challenges/qr-code-component-iux_sIO_H) do [Frontend Mentor](https://www.frontendmentor.io/). O objetivo é reproduzir o cartão com código QR o mais fiel possível ao design, usando HTML e CSS.

## Tecnologias

- HTML5 semântico (`<main>`, hierarquia de títulos)
- CSS3 (layout com **Flexbox**, sombras, `border-radius`, tipografia)
- Fonte do design: **Outfit** (via `font-family` no CSS — confirma que carregas a fonte no projeto, por exemplo com Google Fonts, se ainda não estiver ligada no HTML)

## Como ver localmente

Na pasta do projeto:

```bash
npx serve .
```

Abre o endereço que o comando indicar (por exemplo `http://localhost:3000`) e confere o `index.html` com o `styles.css`.

## O que este projeto inclui

- Cartão centrado no ecrã com fundo conforme o *style guide*
- Imagem do QR com cantos arredondados e texto de apoio (título + parágrafo)
- Estrutura simples e fácil de evoluir (classes em `index.html` / `styles.css`)
